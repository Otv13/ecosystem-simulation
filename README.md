# Ecosystem Simulation (Processing)

## Overview
This project is an interactive ecosystem simulation developed using Processing. It models the interactions between three types of organisms: predators, prey, and plants. The simulation allows users to explore how different variables affect the balance of the ecosystem.

---

## Features
- Predator-prey interaction system
- Plants as a renewable food source
- Real-time simulation updates
- User controls:
  - Pause / Play simulation
  - Reset environment
  - Speed control (slider)
  - Toggle plant growth (checkbox)
- Population counters for predators, prey, and plants
- Click to add plants into the environment

---

## How It Works
- **Predators** chase and consume prey
- **Prey** move around and eat plants
- **Plants** grow over time or can be placed manually by the user

The ecosystem changes dynamically depending on the number of organisms and available resources.

---

## Controls
- **Pause Button** – Stops or resumes the simulation  
- **Reset Button** – Restarts the ecosystem  
- **Slider** – Adjusts simulation speed  
- **Checkbox** – Enables or disables plant growth  
- **Mouse Click** – Adds plants to the environment  

---

## Goal
The goal of the simulation is to observe how predator, prey, and plant populations interact and to experiment with different settings to achieve a balanced ecosystem.
